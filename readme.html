The Searchles Discovery Wordpress Widget is a wrapper that displays the Searchles Discovery Widget in the sidebar of a Wordpress blog on the posts page. Just as you can configure the colors and widget of the Searchles Widget, you can do the same in the wordpress admin tool. Assuming that you have already signed up for Searchles Discovery, To use simply do the following:


  1. In the wordpress admin tool, upload the widget and activate it (Plugins > Add New)
  2. Go to Appearance > Widgets.Â  Searchles Discovery should appear in the widgets
  3. Click on "Add" next to "Searchles Discovery"
  4. In the right-hand side, in the settings of the widget, enter the client ID. Also, you can configure colors and width/height
  5. Now, by default, the widget will not display in the posts page. This isn't because of the widget itself, but because of the fact that the wordpress sidebar does not show up on posts pages. To enable this, you can do the following: (The following is standard wordpress functionality, ignore if you are already familiar with it):
    1. Go to the template that you are using in wordpress in the admin tool
    2. Click on the Main Index Page to change it
    3. Notice that the main div has the class "narrowcolumn", you'll be changing the posts page to this so the sidebar will appear correctly
    4. Go towards the bottom of the file, and select the line: <?php get_sidebar(); ?>
    5. Click on the Single Page in the template
    6. Change the class name from "widecolumn" to "narrowcolumn" in the top dif, usually the first one you see
    7. Go towards the bottom of the page, and above the line that displays the footer, add the code that displays the sidebar (that you copied from the Main Index Page)
